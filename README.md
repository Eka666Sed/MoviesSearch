# MoviesSearch

The application for searching for movies, viewing information about movies and actors while maintaining the search history.

# Stack
- Android Studio
- Clean Arcitecture
- MVVM
- Single Activity
- Android Navigation
- Room
- Retrofit2
- StateFlow
